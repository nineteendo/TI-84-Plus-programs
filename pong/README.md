# Pong

Normal and wall mode

## Normal

Files:

- [**PONG.8xp**](PONG.8xp): TI-84 Plus program
- [**pong.txt**](pong.txt): formatted souce code

Controls:

- <kbd>Up</kbd>: move up
- <kbd>Down</kbd>: move down
- <kbd>Enter</kbd>: (un)pause
