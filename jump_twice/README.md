## Controls

- <kbd>Y=</kbd>: go `2` levels back
- <kbd>Window</kbd>: go `1` level back
- <kbd>Zoom</kbd>: reload level
- <kbd>Trace</kbd>: go `1` level further
- <kbd>Graph</kbd>: go `2` levels further
- <kbd>2nd</kbd>: jump
- <kbd>Enter</kbd>: exit sleep mode
