# Super Simon

## Files

- [**SIMON.8xp**](SIMON.8xp): TI-84 Plus program
- [**super_simon.txt**](super_simon.txt): formatted souce code

## Title Screen Controls

- <kbd>Enter</kbd>: start game

## Controls

- <kbd>Up</kbd>: up
- <kbd>Left</kbd>: left
- <kbd>Right</kbd>: right
- <kbd>Down</kbd>: down
- <kbd>Enter</kbd>: exit sleep mode
