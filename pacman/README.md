## 3. Controls

### 3.1. Title screen

- <kbd>Enter</kbd>: start game

### 3.2. Game

- <kbd>Left</kbd>: move left
- <kbd>Up</kbd>: move up
- <kbd>Right</kbd>: move right
- <kbd>Down</kbd>: move down
- <kbd>Enter</kbd>: (un)pause
